容器的属性

1、flex-direction属性

flex-direction：属性决定主轴的方向（即项目的排列方向）。

    row（默认值）：主轴为水平方向，起点在左端。
    row-reverse：主轴为水平方向，起点在右端。
    column：主轴为垂直方向，起点在上沿。
    column-reverse：主轴为垂直方向，起点在下沿

    .box-1 {
    display: flex;
    flex-direction: row;
    }
    通过如下效果可以看出，子元素的排序方式是从左往右排列

2、flex-wrap属性

默认情况下，项目都排在一条线（又称"轴线"）上。flex-wrap属性定义，如果一条轴线排不下，如何换行。

    nowrap(默认值)      不换行
    wrap               换行，第一行在上方。
    wrap-reverse       换行，第一行在下方。

    .box-5{
            display: flex;
            flex-direction:row; /*默认水平从左往右排列*/
            flex-wrap:nowrap;/*不换行，在一行显示*/
        }
    通过如下效果可以看出，子元素的宽度加起来其实已经超出父容器的宽度，但是，并没有挤出去，而是，自动控制子元素的宽度，
    正好占满父容器宽度，并且在第一行显示，没有因为父容器宽度不够，而挤到第二行，也是由于设置了flex-wrap:nowrap;的原因。

3、flex-flow属性

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap。

    row nowrap;(默认值)     排列从左往右， 不换行
    row wrap                排列从左往右， 换行，第一行在上面
    row wrap-reverse        排列从左往右， 换行，第一行在下面
    column nowrap;(默认值)     排列从左往右， 不换行
    column wrap                排列从左往右， 换行，第一行在上面
    column wrap-reverse        排列从左往右， 换行，第一行在下面

    .box-8{
            display: flex;
            flex-flow:row nowrap;/* 从左往右排列，不换行 */
        }
    通过如下效果可以看出，子元素的宽度加起来其实已经超出父容器的宽度，但是并没有换行，而是所有子元素进行自适应
    并且排列方式是从左往右排列

4、justify-content属性

justify-content属性定义了项目在主轴(水平轴)上的对齐方式。

    flex-start(默认值)     左对齐
    flex-end               右对齐
    center                 居中
    space-between：        两端对齐，项目之间的间隔都相等。
    space-around：         每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。

    .box-10{
            display: flex;
            flex-flow:row nowrap;/*从左往右排列，不换行*/
            justify-content: flex-start;/*左对齐*/
            height:400px;
        }
    通过如下效果可以看出，子元素，从左往右排列，并且不换行
    如果当子元素宽度没有占满父元素的时候，子元素整体在父元素内，左对齐；
    如果当子元素宽度占满父元素的时候，每个子元素整体平均宽度，占满父元素
    内容排列在主轴上，也就是在父元素内的上部分,左对齐
